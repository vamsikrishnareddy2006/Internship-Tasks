# 📊 Student Performance Analysis using Pandas

## 📌 Project Overview

This project demonstrates how to analyze student performance data using the **Pandas** library in Python. It includes data loading, data exploration, statistical analysis, filtering, grouping, and data visualization to gain meaningful insights from the dataset.


## 🎯 Project Objectives

- Load and explore the student performance dataset.
- Perform data cleaning and validation.
- Analyze student grades and study habits.
- Identify trends using statistical analysis.
- Visualize data with charts and graphs.
- Export the processed data for future use.

## 🛠️ Technologies Used

- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📁 Dataset

**Dataset Name:** Student Performance Dataset

The dataset contains information about students, including:

- Age
- Gender
- Study Time
- Failures
- Absences
- Internet Access
- Family Support
- Grades (G1, G2, G3)

## ✨ Features

- Load CSV data using Pandas
- Display dataset information
- Check missing values
- Display statistical summary
- Select and filter data
- Sort data
- Group data using `groupby()`
- Calculate mean, median, minimum, and maximum values
- Find correlations between numerical columns
- Create new columns
- Generate visualizations
- Export processed data to a CSV file

## 📊 Visualizations

This project generates the following charts:

- 📈 Study Time vs Grade
- 📉 Grade Distribution
- 🔥 Correlation Heatmap
